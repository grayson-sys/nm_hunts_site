# Nevada Big Game Draw System -- Deer & Elk

Last updated: 2026-03-04

---

## 1. DRAW SYSTEM MECHANICS

### Draw Type: Bonus Point System with Squared Weighting

Nevada uses a **bonus point** system (not preference points). This is a weighted lottery -- bonus points increase your odds but never guarantee a tag. There is no "max-point" pool that draws first.

### Exact Math of the Weighted Lottery

Every applicant receives random numbers from 1 to 99,000,000. The number of random numbers you receive is:

```
entries = (bonus_points)^2 + 1
```

The "+1" represents your current-year application. Only the **lowest** random number among all your entries is used. Applications are drawn in sequential order starting with the lowest random number.

**Examples:**

| Bonus Points | Entries in Draw | Notes |
|---|---|---|
| 0 | 1 | First-time applicant |
| 1 | 2 | |
| 3 | 10 | |
| 5 | 26 | |
| 10 | 101 | |
| 15 | 226 | |
| 20 | 401 | |
| 33 | 1,090 | Approximate max points as of 2025 |

Even with maximum points, drawing is not guaranteed -- in popular units with few tags, max-point applicants can and do go undrawn.

### Weapon Types and Draw Structure

Nevada does **not** run separate draws for rifle/archery/muzzleloader. Instead, each hunt code specifies a weapon type (Any Legal Weapon, Archery, or Muzzleloader), and applicants list specific hunt codes as their choices. All weapon types are drawn in the same overall draw process.

Legal weapon categories:
- **Any Legal Weapon (ALW):** Centerfire rifle .22-.50 cal (case length <= 3"), shotgun, muzzleloader, or archery
- **Archery:** Longbow, compound bow, or recurved bow
- **Muzzleloader:** Single-barrel muzzleloading firearm, .45 caliber or larger

### Number of Choices per Application

Applicants may select up to **5 hunt choices** per species/category application, listed in order of preference. Each hunt choice is a specific hunt code (unit + weapon type + season). If you are not drawn for choice 1, your application rolls to choice 2, and so on through choice 5.

### Draw Group Order (from CR 26-01)

The Big Game Main Draw is performed by groups, in this order:

1. Silver State
2. Partners in Wildlife
3. Junior Mule Deer Antlered (or Junior Mule Deer Antlered/Antlerless if available)
4. Drawn simultaneously: Rocky Mountain Bighorn Sheep Ram, California Bighorn Sheep Ram, Desert Bighorn Sheep Ram, **Elk Antlered**, Antelope Horns Longer than Ears, **Mule Deer Antlered**, Mountain Goat, Bear, Moose Antlered
5. Drawn simultaneously: Elk Depredation Antlered, Management Bighorn Sheep Ram
6. Drawn simultaneously: California Bighorn Sheep Ewe, Desert Bighorn Sheep Ewe, **Elk Antlerless**, Antelope Horns Shorter than Ears, Junior Antelope Horns Shorter, **Mule Deer Antlerless**
7. Spike Elk
8. Elk Depredation Antlerless

### What Happens If You Don't Draw Any of Your 5 Choices

- You receive **one bonus point** for that species/category (provided you purchased a hunting license).
- Your application fee is **not refunded** (it is non-refundable).
- Your bonus points carry forward and increase by one for the next year.
- If you fail to apply (or purchase a bonus point) for **two consecutive years**, all accumulated bonus points for that species are **forfeited to zero**.

## 2. POINT BANKING / TAG TURN-BACK

### Can Hunters Turn In a Tag After Drawing to Keep/Gain Points?

**Yes.** Nevada allows tag returns with bonus point restoration.

### Rules and Deadlines for Tag Turn-Back

- A tag holder may return their tag to NDOW for **any reason** and receive a restoration of bonus points.
- The return must be **received by NDOW at least one (1) business day before the opening day** of that tag's season.
- All portions of the tag must be included in the return.
- Use the official "Return a Tag" form available at [ndow.org](https://www.ndow.org/wp-content/uploads/2024/05/Return-a-Tag-Form-ADA.pdf).
- After the main draw results, there is also a **7-day electronic return window** during which a successful tag recipient can electronically return the tag prior to printing and mailing (per CR 26-01).

### Effect on Points of Turning Back a Tag

- The applicant is treated **as if the tag had not been issued** and the applicant was unsuccessful -- bonus points are restored to pre-draw levels.
- **Party hunt exception:** All members of a party must return their tags for any member to receive point restoration, unless the return is due to medical, military, or death reasons (with documentation).

### Can You Buy a Point Without Applying for a Hunt?

**Yes.** You can purchase a bonus point without submitting a tag application. Requirements:
- Must have a valid hunting license for that year
- Pay the non-refundable application fee ($10 per species)
- Bonus points can be purchased during any standard big game application period at [ndowlicensing.com](https://www.ndowlicensing.com)
- Only one bonus point can be obtained per season per species/category

### Maximum Bonus Points Currently Held

As of the 2025 draw, the maximum bonus points any individual holds is approximately **33 per species** (the draw system began in 1992). This number increases by one each year, so for the 2026 draw, the theoretical maximum is **~34 points** (yielding 1,157 entries).

## 3. RESIDENT VS NONRESIDENT RULES

### Tag Allocation

- Nonresidents receive approximately **10% of the total tag quota** for each hunt.
- The remaining ~90% goes to residents.
- The exact split can vary slightly depending on final approved quotas.

### Separate Draws or Combined with Quotas?

Nevada runs **separate hunt codes** for residents and nonresidents, each with a set quota. Nonresidents compete only against other nonresidents. Residents compete only against other residents. This is a quota-based separation within the same draw process.

### Restricted Nonresident Guided Mule Deer Hunt

Nevada has a special "Restricted Nonresident Guided Mule Deer Hunt" draw that runs **before** the main draw (deadline: March 9, 2026). These tags must not exceed 16% of nonresident deer tags issued the previous year, or 400 tags, whichever is greater. Any unissued or returned guided tags roll into the main draw nonresident mule deer quota.

### Waiting Periods After Drawing

| Species | Waiting Period |
|---|---|
| Mule Deer (antlered or antlerless) | **None** -- can apply every year |
| Elk Antlered | **7 years** before eligible to apply again |
| Elk Antlerless | **None** |
| Elk Spike | **None** |
| Antelope (buck) | 3 years |
| Bighorn Sheep | 10 years |
| Mountain Goat | 10 years |

### Nonresident Caps

No hard cap on the number of nonresident applicants, but nonresident tag allocation is limited to ~10% of quotas.

### Residency Requirements (NRS 502.015)

To qualify as a Nevada resident for hunting purposes:
- Must be a U.S. citizen or lawfully entitled to remain in the U.S.
- Must have maintained **principal and permanent residence** in Nevada for the **6 months preceding** the application
- Must have been physically present in the state (except temporary absences)
- Must **not** have purchased or applied for hunting/fishing privileges as a resident of another state

**Exceptions:**
- Active military stationed in Nevada qualify immediately (with dependents)
- Full-time students enrolled in a Nevada institution for 6+ months qualify

## 5. OVER-THE-COUNTER (OTC) TAGS

### OTC Deer: No

All deer tags in Nevada are distributed through the draw process. There are no OTC deer tags.

### OTC Elk: No

All elk tags in Nevada are distributed through the draw process. There are no OTC elk tags.

### Mountain Lion: Yes (OTC)

Mountain lion tags may be purchased over the counter without entering a draw.

### Which Units/Zones Are OTC vs Draw?

**All deer and elk hunts in every unit are draw-only.** There are no OTC units for deer or elk.

### Second Draw (Leftover Tags)

After the main draw, remaining tags enter a **Second Draw** (deadline: June 16, 2026; results by June 26, 2026). The second draw:
- Does **not** use bonus points (random lottery only)
- Does **not** award bonus points to unsuccessful applicants
- Is open to both residents and nonresidents, except: junior mule deer and antlerless mule deer tags are **resident-only** in the second draw

### First-Come, First-Served (FCFS) Tags

Tags remaining after the second draw, or returned tags with less than 14 days before season, become available for purchase on a **first-come, first-served basis** at [ndowlicensing.com](https://www.ndowlicensing.com).

## 6. LANDOWNER TAGS

### Landowner Tag Program: Yes

Nevada operates the **Landowner Damage Compensation Tag Program**, established July 1992 under NRS 502.

### How It Works

- Landowners suffering **verified wildlife damage** to agricultural lands may receive compensation tags from NDOW.
- NDOW verifies the damage claim and issues tags accordingly.
- Tags are species-specific: mule deer, pronghorn antelope, and elk tags are available.

### Tags Tied to Property vs Unit-Wide

Tags are issued based on verified damage to specific private land, but the tags themselves are valid for the **unit** in which the property is located.

### Transferable/Saleable?

**Yes.** Landowner compensation tags are **fully transferable**. The landowner, lessee, or manager of the private land may:
- Use the tag themselves (with a valid Nevada hunting license), or
- **Sell the tag** to any holder of a valid Nevada hunting license at **any mutually agreed-upon price**

Market prices are unregulated and driven by supply and demand. Premium unit tags can command substantial prices.

### Volume Relative to Public Draw Tags

Landowner compensation tags are **capped at 2.5% of total tag quotas** per species. This makes them a very small portion of total available tags.

### Additional Landowner Tag Types

- **Elk Incentive Tags** -- issued to landowners in elk management areas
- **Antlerless Elk Landowner Tags** -- for elk damage mitigation
- These may be obtained **in addition to** any tags obtained through the regular draw process

### Bonus Point Tables

NDOW publishes annual bonus point tables that include:
- Breakdown of **successful and total applicants by bonus point level and choice (1st-5th)**
- Separate tables for each species, weapon type, and residency status
- Available for 2022-2025 on the main page; earlier years in the NDOW publications library

Species covered in bonus point tables:
- Mule Deer (ALW, Archery, Muzzleloader -- Resident and Nonresident)
- Elk (Antlered, Antlerless, Spike)
- Antelope, Bighorn Sheep (3 subspecies), Mountain Goat, Black Bear, Moose

### Harvest Reports

| Resource | Details |
|---|---|
| 2024 Big Game Hunt Stats | Downloadable Excel file (filterable) |
| Species-specific harvest PDFs | Mule deer, elk, antelope, bighorn sheep, mountain goat, bear, moose |
| Historical data | 2022-present on main page; pre-2022 in NDOW library ("Other Hunt Statistics" category) |
| Elk-specific data | Harvest stats, antler length data, 50-inch beam counts |
| Deer-specific data | Harvest by unit, composition, four-point analysis |

### Can Data Be Auto-Downloaded?

- The **Excel files** (e.g., 2024 Big Game Hunt Stats) can be directly downloaded via URL from the NDOW website.
- **Bonus point tables** are PDFs that can be downloaded programmatically.
- There is **no public API** for draw data.
- Third-party tools (goHUNT, Huntin' Fool, HuntScore, onX) aggregate and present this data but require subscriptions.
- Direct URL pattern for NDOW PDFs: `https://www.ndow.org/wp-content/uploads/YYYY/MM/filename.pdf`
