--- IDAHO ---
What: Draw odds by hunt unit (controlled hunts, deer + elk)
Where: https://fishandgame.idaho.gov/ifwis/huntplanner/odds/
Steps:
  1. Open the page in a browser
  2. Filter Species = Deer, run export (CSV or Excel), save as:
       ID/raw_data/idaho_deer_draw_odds_2025.csv (or .xlsx)
  3. Repeat for Species = Elk:
       ID/raw_data/idaho_elk_draw_odds_2025.csv (or .xlsx)
  4. If prior years (2023, 2024) are selectable, grab those too
Notes: The export button is in the Hunt Planner tool UI — look for
       a download/export icon near the results table.


--- MONTANA ---
What: Draw statistics for limited-entry deer and elk permits
Where: https://myfwp.mt.gov/fwpPub/drawingStatistics
Steps:
  1. Open the page in a browser
  2. Select Species = Elk, pull stats, export or save as:
       MT/raw_data/montana_elk_draw_statistics_2025.pdf (or .xlsx)
  3. Repeat for Species = Deer:
       MT/raw_data/montana_deer_draw_statistics_2025.pdf (or .xlsx)
  4. Grab 2024 too if the year selector allows it
Notes: This is a myFWP interactive portal. It may require selecting
       the permit type (B-license, special permit, etc.) separately.
       Grab everything you can — any export format is fine.


--- WASHINGTON ---
What: Special permit draw odds (applications vs. successful, by GMU)
Where: WDFW draw results page — https://wdfw.wa.gov/licenses/hunting/game/elk/draw
       (also check https://wdfw.wa.gov/licenses/hunting/game/deer/draw)
Steps:
  1. Open the page in a browser — look for Power BI dashboard links
  2. In the Power BI viewer, click the three dots (...) menu
  3. Select "Export data" → download as CSV or Excel
  4. Save as:
       WA/raw_data/washington_elk_draw_odds_2025.csv (or .xlsx)
       WA/raw_data/washington_deer_draw_odds_2025.csv (or .xlsx)
  5. Repeat for 2024 if available
Notes: Power BI is the only source for WA draw odds — there is no
       direct download link. The export option may say "Export" or
       show a download icon depending on the dashboard version.
       If export is disabled, screenshot the tables as a fallback.
